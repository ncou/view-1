Soli View
--------------

此项目对常用模版引擎进行封装，便于集成到开发者所熟悉的框架。
目前支持以下模版引擎：

- Twig
- Smarty

## 安装

使用 `composer` 安装到你的项目：

    composer require soliphp/view

## 使用

[examples] 目录下提供了 Soli View 的相关示例。

### Twig

安装 Twig 模版引擎：

    composer require twig/twig

运行示例：

    php examples/twig.php

### Smarty

安装 Smarty 模版引擎：

    composer require smarty/smarty

运行示例：

    php examples/smarty.php
